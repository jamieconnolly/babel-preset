> My own opinionated Babel preset.

## Installation

```
$ npm install --save-dev @jamieconnolly/babel-preset
```

## Usage

To get started, add this to your `.babelrc` file:

```json
{
  "presets": ["@jamieconnolly/babel-preset"]
}
```
